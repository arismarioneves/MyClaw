Features Recursos

Everything you need, nothing you don't Tudo que você precisa, nada além disso

Built to be lightweight but powerful where it counts. Leve, mas poderoso onde importa.

Task Scheduling Agendamento de Tarefas
Schedule prompts to run automatically using cron syntax. Daily summaries, weekly reports — set it once, forget it. Agende prompts para rodar automaticamente com sintaxe cron. Resumos diários, relatórios semanais — configure uma vez e esqueça.
Global Memory Memória Global
The bot remembers what matters. Context persists across conversations and decays naturally over time, like a real assistant would. O bot lembra o que importa. O contexto persiste entre conversas e decai naturalmente com o tempo, como um assistente de verdade.
Background Service Serviço em Background
Runs silently as a background service via PM2, launchd, or systemd. Always on — even when your terminal is closed. Roda silenciosamente como serviço via PM2, launchd ou systemd. Sempre ativo — mesmo com o terminal fechado.
Media Support Suporte a Mídia
Send photos and documents directly in chat. Claude analyzes images, reads files, and responds — no extra setup needed. Envie fotos e documentos direto no chat. Claude analisa imagens, lê arquivos e responde — sem configuração extra.
Multi-Messenger Multi-Messenger
Telegram and Slack running simultaneously. Use whichever app you already have open — both work at the same time. Telegram e Slack rodando ao mesmo tempo. Use o app que já está aberto — os dois funcionam simultaneamente.
Fully Customizable Totalmente Personalizável
Edit LIZZ.md to change the assistant's name, personality, tone, and available skills. Changes take effect instantly. Edite LIZZ.md para mudar nome, personalidade, tom e habilidades. As mudanças entram em efeito na próxima mensagem.
